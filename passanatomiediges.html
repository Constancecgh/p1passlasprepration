🔴 Incontournables
Le tube digestif — trajet complet
Bouche → pharynx → œsophage (25 cm, médiastin postérieur) → estomac (cardia, fundus, corps, antre, pylore) → duodénum (C duodénal, rétropéritonéal, reçoit la bile + enzymes pancréatiques à l'ampoule de Vater) → jéjunum → iléon → cæcum (appendice) → côlon (ascendant → transverse → descendant → sigmoïde) → rectum → canal anal.
Ulcère duodénal postérieur : peut éroder l'artère gastroduodénale → hémorragie digestive haute massive (méléna, hématémèse). Cause : H. pylori (80%). Traitement : IPP + amoxicilline + clarithromycine (triple thérapie d'éradication).
Le foie — anatomie et vascularisation
Organe le plus lourd (~1,5 kg), hypochondre droit. Double vascularisation : veine porte (75% du débit, sang riche en nutriments du tube digestif) + artère hépatique (25%, sang oxygéné de l'aorte). Retour sanguin par les veines sus-hépatiques → VCI.
Hile hépatique et voies biliaires
Triade portale : veine porte + artère hépatique + voie biliaire principale.
Canaux hépatiques D et G → canal hépatique commun → canal cholédoque + canal de Wirsung → ampoule de Vater → D2 duodénal.
Anatomie chirurgicale : 8 segments de Couinaud (basée sur la vascularisation).
Cirrhose → hypertension portale → varices œsophagiennes (risque hémorragie massive), ascite, splénomégalie, circulation collatérale abdominale (tête de Méduse).
Le pancréas
Glande mixte rétropéritonéale : tête (dans le C duodénal) + corps + queue (touche la rate). Fonction exocrine : suc pancréatique (amylase, lipase, trypsinogène) → canal de Wirsung → ampoule de Vater. Fonction endocrine : îlots de Langerhans — cellules B (insuline), A (glucagon), D (somatostatine).
Pancréatite aiguë : autodigestion par activation prématurée des enzymes. Causes : alcool (40%) + calculs biliaires (40%). Douleur épigastrique transfixiante irradiant dans le dos, lipase × 3N. Signe de Cullen (ecchymose péri-ombilicale) = pancréatite nécrotico-hémorragique sévère.
Vascularisation digestive
Trois troncs artériels de l'aorte : Tronc cœliaque (T12) → artères hépatique + splénique + gastrique gauche → étage sus-mésocolique (foie, estomac, rate, pancréas). AMS (artère mésentérique supérieure, L1) → grêle + côlon droit. AMI (artère mésentérique inférieure, L3) → côlon gauche + sigmoïde + rectum supérieur.
À retenir
Tronc cœliaque → foie + rate + estomac + pancréas · AMS → grêle + côlon droit · AMI → côlon gauche + sigmoïde + rectum
La veine porte = confluence veine splénique + veine mésentérique supérieure → retour veineux digestif vers le foie
Ischémie mésentérique aiguë (occlusion AMS, souvent par embolie cardiaque) → urgence chirurgicale : douleur abdominale intense + défense + diarrhée sanglante + choc septique si nécrose. Mortalité >50% si prise en retard. Angio-scanner + revascularisation.
🟡 Importants
Le péritoine
Feuillet pariétal (tapisse la paroi abdominale) + feuillet viscéral (entoure les organes). Grande cavité péritonéale + arrière-cavité des épiploons (communicants par le foramen de Winslow). Mésentère = repli reliant le grêle à la paroi postérieure.
Péritonite = inflammation du péritoine par perforation d'un organe creux (ulcère perforé, appendicite perforée). Clinique : ventre de bois (contracture), douleur généralisée, fièvre, sepsis. Radio : pneumopéritoine (croissant gazeux sous le diaphragme). Urgence chirurgicale.
La vésicule biliaire et les voies biliaires
La vésicule biliaire stocke et concentre la bile. La bile est libérée lors des repas dans le duodénum via le canal cholédoque (confluence canal cystique + canal hépatique commun). Le sphincter d'Oddi régule le flux à l'ampoule de Vater.
Triade de Charcot (douleur biliaire + fièvre + ictère) = angiocholite (calcul dans le cholédoque + infection biliaire). Urgence : antibiotiques IV + drainage biliaire (CPRE). Calcul bloquant à l'ampoule de Vater → pancréatite aiguë biliaire.
La rate
Organe lymphoïde vascularisé par l'artère splénique (branche du tronc cœliaque). Fonctions : filtration sanguine (destruction des GR vieillis), réservoir de plaquettes, immunité (production d'anticorps). Non vitale — splénectomie possible mais impose vaccination anti-pneumocoque, méningocoque, Haemophilus.
Rupture splénique post-traumatique → hémopéritoine → choc hémorragique. La rate est protégée par les côtes 9-11 gauches. Mononucléose infectieuse (EBV) → splénomégalie → contre-indication aux sports de contact (risque de rupture spontanée).
🔵 Base
Paroi digestive et plexus nerveux entérique
4 couches (de l'intérieur vers l'extérieur) : muqueuse (épithélium + chorion + musculaire muqueuse) → sous-muqueuse (vaisseaux + plexus de Meissner) → musculeuse (circulaire interne + longitudinale externe + plexus d'Auerbach) → séreuse (péritoine viscéral).
Maladie de Hirschsprung = absence des cellules ganglionnaires du plexus d'Auerbach dans le recto-sigmoïde → pas de péristaltisme → mégacôlon congénital. Clinique néonatale : retard du méconium >48h + ballonnement. Traitement chirurgical.
La vascularisation digestive (tronc cœliaque, AMS, AMI) et la double vascularisation du foie (veine porte + artère hépatique) sont parmi les notions les plus testées en QCM. Mémoriser qui irrigue quoi te fait gagner des points facilement.
