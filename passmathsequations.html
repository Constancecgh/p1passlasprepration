Équations du 2ème degré — discriminant
L'équation ax² + bx + c = 0 est une équation du 2ème degré. Sa résolution passe par le calcul du discriminant Δ = b² - 4ac.
Résolution selon le signe de Δ
Δ = b² - 4ac

Si Δ > 0 : deux solutions distinctes :
x₁ = (-b + √Δ) / (2a) et x₂ = (-b - √Δ) / (2a)

Si Δ = 0 : une solution double :
x₁ = x₂ = -b/(2a)

Si Δ < 0 : pas de solution réelle

Relations de Viète (utiles pour vérifier) :
— x₁ + x₂ = -b/a
— x₁ × x₂ = c/a

Factorisation : ax² + bx + c = a(x - x₁)(x - x₂) quand Δ ≥ 0
Résoudre 2x² - 5x + 3 = 0 : Δ = 25 - 24 = 1 > 0. x₁ = (5+1)/4 = 3/2, x₂ = (5-1)/4 = 1. Vérification Viète : x₁+x₂ = 5/2 = -(-5)/2 ✓, x₁×x₂ = 3/2 = 3/2 ✓. En chimie : les équilibres chimiques conduisent souvent à des équations du 2ème degré (calcul de la concentration à l'équilibre en fonction de Ka).
Inéquations et tableaux de signes
Une inéquation est une inégalité (< ou >) à résoudre. La méthode consiste à ramener l'inéquation à une étude de signe d'une expression.
Méthode du tableau de signes
Étape 1 : Tout mettre d'un côté (≥ 0 ou ≤ 0)
Étape 2 : Factoriser si possible
Étape 3 : Identifier les racines (là où chaque facteur = 0)
Étape 4 : Tableau de signes pour chaque facteur
Étape 5 : Signe du produit = produit des signes

Règles de signe :
— (+)×(+) = + · (-)×(-) = + · (+)×(-) = -
— 1/positif = positif · 1/négatif = négatif

Attention aux cas particuliers : quand on multiplie ou divise par un négatif, le sens de l'inégalité s'inverse !
Résoudre (x-2)(x+3) ≥ 0 : racines en x=2 et x=-3. Tableau : pour x < -3, les deux facteurs sont négatifs → produit positif ✓. Pour -3 < x < 2 : facteurs de signes opposés → produit négatif ✗. Pour x > 2 : deux positifs → positif ✓. Solution : x ≤ -3 ou x ≥ 2.
Systèmes d'équations linéaires
Un système d'équations linéaires est un ensemble d'équations du 1er degré à résoudre simultanément. En PASS, ils apparaissent en chimie (mélanges, conservation), en physique (circuits, forces) et en biostatistiques.
Méthodes de résolution
Substitution : isoler une variable dans une équation et la substituer dans l'autre.

Addition/soustraction (élimination) : multiplier les équations pour éliminer une variable.

Méthode de Cramer (2×2) :
Pour ax + by = e et cx + dy = f :
x = (ed - bf)/(ad - bc) et y = (af - ec)/(ad - bc)
avec D = ad - bc (déterminant, ≠ 0 pour solution unique)

3 cas possibles :
— D ≠ 0 : une unique solution
— D = 0 et équations proportionnelles : infinité de solutions
— D = 0 et équations non proportionnelles : aucune solution
En chimie, mélange : 'x litres de solution A à 20% et y litres de solution B à 50% pour obtenir 10 litres à 35%'. Système : x + y = 10 et 0,2x + 0,5y = 3,5. De la 1ère : y = 10-x. Substitution : 0,2x + 0,5(10-x) = 3,5 → 0,2x + 5 - 0,5x = 3,5 → -0,3x = -1,5 → x = 5. Donc x = 5 L et y = 5 L.

Puissances et racines — rappels
Les puissances et racines sont omniprésentes en sciences. Il est indispensable de maîtriser leurs règles de calcul.
Règles de calcul
Puissances :
— aᵐ × aⁿ = aᵐ⁺ⁿ
— aᵐ / aⁿ = aᵐ⁻ⁿ
— (aᵐ)ⁿ = aᵐⁿ
— a⁰ = 1 (pour a ≠ 0)
— a⁻ⁿ = 1/aⁿ
— a^(1/n) = ⁿ√a (racine n-ième de a)

Notation scientifique : a × 10ⁿ (indispensable en sciences)
Ex : 6,022 × 10²³ (nombre d'Avogadro)

Racines :
— √(a×b) = √a × √b
— √(a/b) = √a / √b
— (√a)² = a (pour a ≥ 0)
En physique : la loi de Coulomb F = k·q₁·q₂/r². Si r double, F est divisée par 4 (1/2² = 1/4). En chimie : la constante d'équilibre K = [produits]/[réactifs] → les puissances correspondent aux coefficients stœchiométriques. En biologie : la population double : N(t) = N₀ × 2^(t/T) où T est le temps de doublement.
Proportionnalité et pourcentages
La proportionnalité et les pourcentages sont les outils mathématiques les plus utilisés en pratique médicale — dosages, concentrations, statistiques.
Formules clés
Pourcentage : p% de a = (p/100) × a
Variation en % : (valeur finale - valeur initiale) / valeur initiale × 100
Règle de trois : si a → b alors x → (b×x)/a

Concentrations (chimie et biologie) :
— Concentration molaire [C] = n/V (mol/L)
— Concentration massique Cm = m/V (g/L)
— Dilution : C₁V₁ = C₂V₂

Dose (médicaments) :
— Dose = concentration × volume
— mg/kg : dose × poids du patient
Dilution : préparer 100 mL d'une solution à 0,1 mol/L à partir d'une solution mère à 1 mol/L. C₁V₁ = C₂V₂ → 1 × V₁ = 0,1 × 100 → V₁ = 10 mL. Prélever 10 mL de solution mère et compléter à 100 mL. En médecine : si un patient de 70 kg doit recevoir 5 mg/kg d'antibiotique, la dose est 5 × 70 = 350 mg.
Discriminant Δ = b²-4ac → solutions du 2ème degré. Tableau de signes pour les inéquations. Et C₁V₁ = C₂V₂ pour les dilutions — la formule la plus utilisée en chimie pratique.
