Axe hypothalamo-hypophysaire — régulation hormonale
L'hypothalamus contrôle l'hypophyse via des hormones de libération (RH) et d'inhibition (IH) qui circulent dans le système porte hypothalamo-hypophysaire. L'hypophyse antérieure (adénohypophyse) sécrète les hormones trophes (TSH, FSH, LH, ACTH, GH, prolactine). L'hypophyse postérieure (neurohypophyse) libère l'ADH et l'ocytocine.
Axes hormonaux à connaître
Axe thyroïdien : TRH (hypothalamus) → TSH (hypophyse) → T3/T4 (thyroïde) → rétrocontrôle négatif sur TRH et TSH.
Axe corticosurrénalien : CRH → ACTH → cortisol → rétrocontrôle négatif. Cortisol : anti-inflammatoire, hyperglycémiant, catabolisant protéique, rétention Na+.
Axe gonadotrope : GnRH (pulsatile) → FSH + LH → œstrogènes/testostérone → rétrocontrôle. GnRH continue → down-régulation (base des agonistes GnRH = castration chimique).
Axe somatotrope : GHRH → GH → IGF-1 (foie) → croissance. Somatostatine inhibe GH.
À retenir
Rétrocontrôle négatif : hormone périphérique ↑ → inhibe hypothalamus + hypophyse → ↓ RH et trophe
Exception : LH surge (ovulation) = rétrocontrôle POSITIF des œstrogènes sur LH → pic ovulatoire
Le syndrome de Cushing peut être ACTH-dépendant (adénome hypophysaire = maladie de Cushing, 70% des cas) ou ACTH-indépendant (adénome surrénalien, 15%) ou paranéoplasique (CBPC → ACTH ectopique). Clinique : obésité facio-tronculaire, vergetures pourpres, faiblesse musculaire, HTA, diabète, ostéoporose. Exploration : cortisol libre urinaire + test de freinage à la dexaméthasone + imagerie.
Thyroïde — synthèse et effets des hormones thyroïdiennes
Les hormones thyroïdiennes (T3 = triiodothyronine et T4 = thyroxine) régulent le métabolisme basal, la croissance et le développement du système nerveux. La T4 est la prohormone majoritairement sécrétée, convertie en T3 active dans les tissus périphériques (déiodase).
Hypothyroïdie vs Hyperthyroïdie — tableaux cliniques
Hypothyroïdie : TSH ↑, T4L ↓. Signes : bradycardie, frilosité, prise de poids, fatigue, constipation, myxœdème, dépression. Cause principale : thyroïdite de Hashimoto (anti-TPO, anti-Tg). Traitement : lévothyroxine.
Hyperthyroïdie : TSH ↓, T4L ↑. Signes : tachycardie (FA possible), amaigrissement, nervosité, tremblements, thermophobie, diarrhée, exophtalmie (Basedow). Cause principale : maladie de Basedow (TSH-Rac = TRAb). Traitement : antithyroïdiens (carbimazole, PTU), IRA, chirurgie.
Chez le nouveau-né, l'hypothyroïdie congénitale (TSH effondrée si centrale, TSH élevée si périphérique) cause un retard mental grave et un nanisme si non traitée → dépistage systématique J3 par test de Guthrie (dosage TSH). Le traitement doit être débuté dans les 2 semaines de vie pour préserver le développement neurologique.
Diabète — physiopathologie et régulation glycémique
Le glucose sanguin est régulé entre 0,7 et 1,1 g/L à jeun. L'insuline (cellules β des ilots de Langerhans) est l'hormone hypoglycémiante principale. Le glucagon (cellules α) est l'hormone hyperglycémiante principale.
Diabète de type 1 vs type 2
DT1 : destruction auto-immune des cellules β → déficit absolu en insuline → hyperglycémie + corps cétoniques (DKA si décompensation). Traitement : insulinothérapie obligatoire. Auto-Ac : anti-GAD, anti-IA2, anti-insuline.
DT2 : insulinorésistance + déficit relatif en insuline. Obésité, sédentarité. Traitement : metformine (1ère ligne) + SGLT2i + GLP-1 RA + insuline si nécessaire. HbA1c objectif < 7%.
Critères diagnostiques : glycémie à jeun ≥ 1,26 g/L à 2 reprises · Glycémie aléatoire ≥ 2 g/L avec symptômes · HGPO ≥ 2 g/L · HbA1c ≥ 6,5%.
L'acidocétose diabétique (DKA = urgence du diabète de type 1 : manque d'insuline → cétogenèse → acidose) est une urgence du DT1 : insuline absente → lipolyse ↑↑ → cétogenèse ↑↑ → acétoacétate + β-hydroxybutyrate → acidose métabolique + hyperglycémie + déshydratation. Traitement : insuline IV + soluté salé isotonique + KCl (corriger hypokaliémie avant insuline). Surveillance : glycémie/h, K+/2h, GDS/4h.
🟡 Importants
Axe corticosurrénalien — cortisol et ACTH
La surrénale produit dans son cortex : le cortisol (zona fasciculata, contrôlé par ACTH), l'aldostérone (zona glomerulosa, contrôlée par le SRAA et le K+) et les androgènes surrénaliens (DHEA, zona reticularis). La médullosurrénale produit les catécholamines (adrénaline, noradrénaline).
L'insuffisance surrénalienne aiguë (crise addisonnienne) est une urgence vitale : déficit brutal en cortisol + aldostérone → hypotension réfractaire, hypoglycémie, hyponatrémie, hyperkaliémie, vomissements. Causes : arrêt brutal d'une corticothérapie prolongée (atrophie surrénalienne) ou sepsis sur IS chronique. Traitement : hémisuccinate d'hydrocortisone 100 mg IV URGENT + soluté salé + glucose.
🔵 Base
Régulation du métabolisme phospho-calcique
La calcémie normale : 2,2-2,6 mmol/L (ou 88-104 mg/L). Elle est régulée par la PTH (parathormone : ↑ calcémie → ↑ réabsorption osseuse, ↑ réabsorption rénale, ↑ 1,25(OH)2D3), la vitamine D active (1,25(OH)2D3 = calcitriol : ↑ absorption intestinale Ca2+) et la calcitonine (↑ calcémie → calcitonine → ↓ réabsorption osseuse).
À retenir
PTH ↑ si Ca2+ ↓ · PTH ↓ si Ca2+ ↑ (récepteur calcium-sensing CaSR des cellules principales).
Hypercalcémie : PTH ↑ → hyperparathyroïdie. PTH ↓ + Ca2+ ↑ → cancer (PTHrP ou métastases osseuses).
L'hyperparathyroïdie primitive (adénome parathyroïdien dans 85% des cas) → PTH élevée → hypercalcémie. Les manifestations (règle des 8 « C » en anglais) : calculs rénaux, constipation, douleurs osseuses, confusion, calcifications, complications cardiaques. Traitement : chirurgie (parathyroïdectomie). Les bisphosphonates traitent l'hypercalcémie maligne (métastases osseuses, myélome → ostéoclastes activés).
L'axe thyroïdien est le plus testé : TSH ↑ + T4L ↓ = hypothyroïdie périphérique. TSH ↓ + T4L ↑ = hyperthyroïdie. Et pour le calcium : PTH ↑ + Ca2+ ↑ = hyperparathyroïdie. PTH ↓ + Ca2+ ↑ = cancer. Ces règles de lecture se retrouvent dans presque tous les QCM de physiologie endocrinienne.
