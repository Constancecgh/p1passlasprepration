Cercle trigonométrique et valeurs remarquables
La trigonométrie étudie les relations entre angles et longueurs. Le cercle trigonométrique (cercle de rayon 1 centré à l'origine) est la référence. Les angles sont exprimés en radians (unité utilisée en sciences) ou en degrés.
Conversions et valeurs remarquables
Conversion degrés ↔ radians : 180° = π rad
— 0° = 0 · 30° = π/6 · 45° = π/4 · 60° = π/3 · 90° = π/2 · 180° = π · 360° = 2π

Valeurs à connaître par cœur :
| Angle | sin | cos | tan |
| 0° (0) | 0 | 1 | 0 |
| 30° (π/6) | 1/2 | √3/2 | 1/√3 |
| 45° (π/4) | √2/2 | √2/2 | 1 |
| 60° (π/3) | √3/2 | 1/2 | √3 |
| 90° (π/2) | 1 | 0 | indéfini |

Astuce mnémotechnique pour sin : 0, 1/2, √2/2, √3/2, 1 (de 0° à 90°)
Calculer sin(5π/6) : 5π/6 = π - π/6, donc sin(5π/6) = sin(π/6) = 1/2 (symétrie par rapport à π/2). Calculer cos(3π/4) : 3π/4 = π - π/4, donc cos(3π/4) = -cos(π/4) = -√2/2. Ces calculs reviennent en physique des ondes et en chimie quantique.
Relations trigonométriques fondamentales
Quelques relations liant sin, cos et tan sont indispensables et reviennent constamment dans les problèmes de physique.
Relations à connaître
Relation de Pythagore : sin²(x) + cos²(x) = 1 — la plus importante !
Tangente : tan(x) = sin(x)/cos(x)
Parité : sin(-x) = -sin(x) (impaire), cos(-x) = cos(x) (paire)
Périodicité : sin(x + 2π) = sin(x), cos(x + 2π) = cos(x)
Formules de décalage :
— sin(π - x) = sin(x)
— cos(π - x) = -cos(x)
— sin(π/2 - x) = cos(x)
— cos(π/2 - x) = sin(x)

Dérivées : (sin x)' = cos x · (cos x)' = -sin x
Formule clé
sin²(x) + cos²(x) = 1
La relation de Pythagore trigonométrique — valable pour tout angle x
En physique des ondes : une onde s'écrit y(t) = A·sin(2πft + φ). La puissance moyenne est proportionnelle à A²· = A²/2 (car la moyenne de sin² = 1/2). En biophysique : les potentiels d'action et les oscillations biologiques sont modélisés par des fonctions sinusoïdales.
Formules d'addition et double angle
Les formules d'addition permettent de calculer sin(a+b) et cos(a+b) à partir de sin(a), cos(a), sin(b), cos(b). Les formules du double angle en découlent directement.
Formules d'addition et double angle
sin(a + b) = sin(a)cos(b) + cos(a)sin(b)
sin(a - b) = sin(a)cos(b) - cos(a)sin(b)
cos(a + b) = cos(a)cos(b) - sin(a)sin(b)
cos(a - b) = cos(a)cos(b) + sin(a)sin(b)

Double angle (a = b dans les formules d'addition) :
— sin(2a) = 2·sin(a)·cos(a)
— cos(2a) = cos²(a) - sin²(a) = 2cos²(a)-1 = 1-2sin²(a)

Linéarisation (très utile en physique) :
— cos²(a) = (1 + cos(2a))/2
— sin²(a) = (1 - cos(2a))/2
En optique (interférences) : l'intensité résultante de deux ondes dépend de cos(Δφ) où Δφ est la différence de phase. Avec les formules d'addition : si deux ondes de même amplitude A interfèrent, l'amplitude résultante vaut 2A·cos(Δφ/2). Maximum d'intensité quand Δφ = 0 (interférences constructives).
🔵 Base
Équations trigonométriques simples
Résoudre une équation comme sin(x) = a ou cos(x) = a revient à trouver tous les angles x qui vérifient cette relation sur le cercle trigonométrique.
Solutions des équations de base
sin(x) = a (avec |a| ≤ 1) :
→ x = arcsin(a) + 2kπ ou x = π - arcsin(a) + 2kπ (k entier)

cos(x) = a (avec |a| ≤ 1) :
→ x = ±arccos(a) + 2kπ (k entier)

tan(x) = a :
→ x = arctan(a) + kπ (k entier)

Cas particuliers fréquents :
— sin(x) = 0 → x = kπ
— cos(x) = 0 → x = π/2 + kπ
— sin(x) = 1 → x = π/2 + 2kπ
— cos(x) = 1 → x = 2kπ
sin(x) = 1/2 → x = π/6 + 2kπ ou x = 5π/6 + 2kπ. Sur [0, 2π] : x = π/6 ≈ 0,52 rad et x = 5π/6 ≈ 2,62 rad. En physique : résoudre sin(ωt) = 0,5 pour trouver les instants où l'onde atteint la moitié de son amplitude maximale.
Trigonométrie dans les triangles
Dans un triangle rectangle, les fonctions trigonométriques sont définies comme des rapports de côtés. Ces définitions sont la base de toute la trigonométrie.
Dans un triangle rectangle (angle θ au sommet A)
Adjacent = côté adjacent à θ · Opposé = côté opposé à θ · Hypoténuse = côté face à l'angle droit

sin(θ) = Opposé / Hypoténuse
cos(θ) = Adjacent / Hypoténuse
tan(θ) = Opposé / Adjacent = sin(θ)/cos(θ)

Mnémotechnique : SOH-CAH-TOA
Sin = Opposé/Hypoténuse · Cos = Adjacent/Hypoténuse · Tan = Opposé/Adjacent

Théorème de Pythagore : a² + b² = c² (a, b = côtés, c = hypoténuse)
En biophysique : la résolution d'un microscope optique dépend de sin(θ) où θ est le demi-angle d'ouverture de l'objectif (ouverture numérique NA = n·sin(θ)). En biomécanique : les forces musculaires et les réactions articulaires se calculent avec les fonctions trigonométriques dans les triangles de forces.
Les 3 choses à avoir par cœur : sin²+cos²=1, les valeurs à 0/30/45/60/90°, et (sin u)'=u'·cos u. En PASS : trigonométrie surtout utile en physique des ondes et biophysique.
