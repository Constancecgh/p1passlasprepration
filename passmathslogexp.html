La fonction exponentielle eˣ
La fonction exponentielle eˣ (ou exp(x)) est l'une des fonctions les plus importantes en sciences. Elle est définie pour tout x réel, toujours positive, et est la seule fonction égale à sa propre dérivée.
Propriétés fondamentales de eˣ
Toujours positive : eˣ > 0 pour tout x
e⁰ = 1 (passage par 1 en x=0)
Dérivée : (eˣ)' = eˣ (unique !)
Croissante sur ℝ, tend vers +∞ quand x→+∞, tend vers 0 quand x→-∞

Propriétés algébriques (à connaître par cœur) :
— eᵃ × eᵇ = eᵃ⁺ᵇ
— eᵃ / eᵇ = eᵃ⁻ᵇ
— (eᵃ)ᵇ = eᵃᵇ
— e⁻ˣ = 1/eˣ
— e = 2,718... (constante d'Euler)
Formule clé
eᵃ × eᵇ = eᵃ⁺ᵇ · eᵃ / eᵇ = eᵃ⁻ᵇ · (eᵃ)ᵇ = eᵃᵇ
La base e = 2,718... · eˣ est toujours > 0 · (eˣ)' = eˣ
En pharmacologie : la concentration d'un médicament suit C(t) = C₀·exp(-kt). Quand t augmente, exp(-kt) diminue → le médicament s'élimine. La vitesse d'élimination est proportionnelle à la concentration restante. C'est une cinétique d'ordre 1 — la plus fréquente en pharmacologie.
Le logarithme naturel ln(x)
Le logarithme naturel ln(x) est la fonction réciproque de eˣ. C'est-à-dire que ln(eˣ) = x et exp(ln x) = x. Il est défini uniquement pour x > 0.
Propriétés fondamentales de ln(x)
Défini pour x > 0 uniquement
ln(1) = 0, ln(e) = 1
Dérivée : (ln x)' = 1/x
Croissant, tend vers +∞ lentement, tend vers -∞ quand x→0⁺

Propriétés algébriques (à connaître par cœur) :
— ln(a × b) = ln(a) + ln(b)
— ln(a / b) = ln(a) - ln(b)
— ln(aⁿ) = n·ln(a)
— ln(1/a) = -ln(a)
— ln(√a) = ln(a)/2
Formule clé
ln(a×b) = ln a + ln b · ln(a/b) = ln a − ln b · ln(aⁿ) = n·ln a
Réciproque : ln(eˣ) = x et exp(ln x) = x · ln(1) = 0 · ln(e) = 1
Le pH = -log₁₀[H⁺] utilise le logarithme (en base 10, pas en base e). Relation : log₁₀(x) = ln(x)/ln(10) ≈ ln(x)/2,303. La loi de Beer-Lambert : A = ε·l·c utilise également les logarithmes (absorbance). En cinétique chimique d'ordre 1 : ln([A]) = ln([A]₀) - k·t → relation linéaire entre ln(concentration) et le temps.
Résoudre des équations avec ln et exp
Savoir résoudre des équations faisant intervenir eˣ ou ln(x) est essentiel en chimie, physique et biostatistiques. La clé : utiliser la réciproque pour isoler l'inconnue.
Méthodes de résolution
Équation du type eˣ = a (avec a > 0) :
→ x = ln(a) [appliquer ln des deux côtés]

Équation du type ln(x) = b :
→ x = eᵇ [appliquer exp des deux côtés]

Équation du type eˣ = eᵃ :
→ x = a [l'exponentielle est injective]

Équation du type ln(f(x)) = ln(g(x)) :
→ f(x) = g(x) [ln est injective] et vérifier que f(x) > 0

Astuce : toujours vérifier le domaine de définition après résolution (ln impose x > 0)
Résoudre exp(2x-1) = 5 : appliquer ln → 2x-1 = ln(5) → x = (1+ln(5))/2 ≈ (1+1,609)/2 ≈ 1,305. Résoudre ln(x+2) = 3 : appliquer exp → x+2 = e³ → x = e³-2 ≈ 20,09-2 ≈ 18,09. Vérifier : x+2 = e³ > 0 ✓. En biologie : résoudre N(t) = N₀·exp(kt) pour t quand on cherche le temps de doublement : 2N₀ = N₀·exp(kt) → exp(kt) = 2 → t = ln(2)/k.
🟡 Importants
Applications en sciences — pH, cinétiques, loi de Beer-Lambert
Le logarithme et l'exponentielle sont omniprésents en chimie et physique du PASS. Voici les applications directes les plus importantes.
Applications directes
pH : pH = -log₁₀[H⁺]. Si pH = 7 → [H⁺] = 10⁻⁷ mol/L. Chaque unité de pH = facteur 10 sur la concentration en H⁺.

Cinétique d'ordre 1 : [A](t) = [A]₀·exp(-kt). Demi-vie t½ = ln(2)/k ≈ 0,693/k. La concentration diminue exponentiellement.

Loi de Beer-Lambert : A = ε·l·c où A = -log₁₀(T) = log₁₀(I₀/I). Utilisée pour doser les protéines (DO 280 nm), l'ADN (DO 260 nm).

Potentiel de Nernst (notion) : E = E° + (RT/nF)·ln([ox]/[red]). Le potentiel d'électrode dépend du logarithme du rapport des concentrations.
Si la demi-vie d'un médicament est de 4 heures, quelle fraction reste après 12 heures ? Après 3 demi-vies (12h = 3×4h) : fraction restante = (1/2)³ = 1/8 = 12,5%. Formule générale : après n demi-vies, il reste (1/2)ⁿ. Même raisonnement pour la radioactivité — après 10 demi-vies, il reste 1/2¹⁰ ≈ 0,1% de la quantité initiale.
🔵 Base
Logarithme décimal log₁₀ vs logarithme naturel ln
Il existe deux logarithmes principaux : le logarithme décimal (log₁₀, noté log) utilisé en chimie (pH, dB) et le logarithme naturel (ln, base e) utilisé en mathématiques et physique. La conversion est simple.
Conversion et différences
ln(x) = log₁₀(x) × ln(10) avec ln(10) ≈ 2,303
log₁₀(x) = ln(x) / ln(10) ≈ ln(x) / 2,303

log₁₀ : utilisé pour pH, pKa, décibels, magnitude d'un séisme
ln : utilisé en mathématiques (dérivées, intégrales), cinétiques

Propriétés communes (valables pour log₁₀ et ln) :
— log(a×b) = log(a) + log(b)
— log(a/b) = log(a) - log(b)
— log(aⁿ) = n·log(a)
pH = -log₁₀[H⁺] = -ln[H⁺]/2,303. Si [H⁺] = 10⁻⁴ M : pH = -log₁₀(10⁻⁴) = 4. Si [H⁺] = 5×10⁻⁴ M : pH = -log₁₀(5×10⁻⁴) = -(log 5 + log 10⁻⁴) = -(0,699 - 4) = 3,30. Note : log₁₀(2) ≈ 0,301, log₁₀(5) ≈ 0,699, log₁₀(3) ≈ 0,477 — ces valeurs sont utiles à connaître.
Les 3 réflexes : eˣ × eʸ = eˣ⁺ʸ, ln(xy) = ln x + ln y, exp(ln x) = x. Et pour le PASS : pH = -log[H⁺], demi-vie = ln(2)/k, cinétique d'ordre 1 : C(t) = C₀·exp(-kt).
