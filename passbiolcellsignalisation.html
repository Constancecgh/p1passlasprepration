Récepteurs membranaires et voies de signalisation
La signalisation cellulaire permet aux cellules de communiquer et de répondre à leur environnement. Un ligand (hormone, facteur de croissance, neurotransmetteur) se fixe sur un récepteur → transduction du signal → réponse cellulaire. On distingue 4 grandes familles de récepteurs membranaires.
Les 4 familles de récepteurs membranaires
Récepteurs couplés aux protéines G (RCPG) : 7 domaines transmembranaires, couplés à une protéine G hétérotrimérique (Gαβγ). Gαs → AC → AMPc → PKA. Gαi → AC inhibée → ↓ AMPc. Gαq → PLC → IP3 + DAG → PKC + Ca2+.
Récepteurs tyrosine kinase (RTK) : EGF-R, IGF1-R, VEGFR, PDGFR, c-KIT, RET. Dimérisation → autophosphorylation → cascade RAS/RAF/MEK/ERK ou PI3K/AKT/mTOR.
Récepteurs à activité sérine/thréonine kinase : TGF-β → Smad → régulation transcriptionnelle (différenciation, cicatrisation).
Récepteurs canaux (ionotropiques) : récepteur AMPA, NMDA, GABA-A, nicotinique à ACh → ouverture directe d'un canal ionique → réponse rapide.
Le trastuzumab (Herceptin) est un anticorps monoclonal qui bloque HER2 (RTK suramplifi) → inhibe la cascade RAS/MAPK → arrêt de la prolifération des cellules HER2+. Le sunitinib inhibe plusieurs RTK (VEGFR, PDGFR, c-KIT) → antiangiogénique + antiprolifératif → cancer du rein, tumeurs stromales gastriques (GIST). L'imatinib inhibe c-KIT → GIST.
La voie AMPc — PKA et régulation intracellulaire
L'AMPc (adénosine monophosphate cyclique) est un second messager produit par l'adénylyl cyclase (AC) en réponse aux signaux activant Gαs. Il active la protéine kinase A (PKA) qui phosphoryle de nombreuses cibles.
Voie AMPc → PKA — exemples
Adrénaline + récepteur β : Gαs → AC ↑ → AMPc ↑ → PKA ↑ → phosphorylase kinase → phosphorylase active → glycogénolyse (foie + muscle). Aussi → phosphorylation des canaux Ca2+ cardiaque → inotrope positif.
Glucagon : même voie dans le foie → glycogénolyse + néoglucogenèse.
TSH : Gαs → AMPc → PKA → synthèse T3/T4.
PTH : Gαs → AMPc → PKA → réabsorption rénale Ca2+, réabsorption osseuse.
La phosphodiestérase (PDE) hydrolyse l'AMPc → arrêt du signal. Inhibiteurs PDE (caféine, sildénafil/Viagra sur PDE5 dans les corps caverneux → ↑ GMPc → érection.
La toxine cholérique (Vibrio cholerae) ADP-ribosyle Gαs → Gαs bloqué en état actif → AC constitutivement active → AMPc ↑↑↑ → PKA active en permanence → canal CFTR ouvert en permanence → Cl− sécrété → Na+ et H2O suivent → diarrhée cholérique profuse (jusqu'à 20 L/jour) → déshydratation sévère. Traitement : réhydratation orale (profite du cotransporteur Na+/glucose = SGLT1 préservé).
La voie IP3/DAG — Ca2+ et PKC
Les récepteurs couplés à Gαq activent la phospholipase C (PLC) qui hydrolyse le PIP2 (phosphatidylinositol 4,5-bisphosphate) membranaire en IP3 et DAG. Ces deux seconds messagers agissent en synergie.
IP3 et DAG — effets
IP3 (inositol 1,4,5-trisphosphate) : diffuse dans le cytoplasme → se fixe sur les récepteurs IP3R du RE → libération du Ca2+ stocké dans le RE → [Ca2+] cytoplasmique ↑↑ (de 100 nM à 1-10 µM) → activation de la calmoduline → calcineurine → NFAT → transcription (lymphocytes T) → contraction musculaire → exocytose → prolifération.
DAG (diacylglycérol) : reste dans la membrane → active la PKC (protéine kinase C) → phosphorylation de nombreuses cibles → prolifération, différenciation.
Synergie : IP3 (Ca2+) + DAG → PKC maximalement activée (besoin des deux).
Les immunosuppresseurs inhibent la voie IP3/Ca2+/calcineurine dans les lymphocytes T. La ciclosporine et le tacrolimus inhibent la calcineurine → NFAT ne peut pas aller dans le noyau → pas d'IL-2 → pas de prolifération des lymphocytes T → pas de rejet de greffe. Effets secondaires : néphrotoxicité (vasoconstriction rénale), neurotoxicité.
🟡 Importants
La voie RAS/MAPK et la voie PI3K/AKT/mTOR
Les RTK activent principalement deux grandes cascades de signalisation : la voie RAS/RAF/MEK/ERK (prolifération) et la voie PI3K/AKT/mTOR (survie et croissance). Ces voies sont fréquemment dérégulées dans les cancers.
Les 2 voies oncogènes majeures
RAS/MAPK : RTK activé → GRB2/SOS → RAS-GDP → RAS-GTP → RAF → MEK → ERK → noyau → transcription de gènes de prolifération (cyclines, c-Fos, c-Jun). Oncogènes fréquents : KRAS, BRAF (V600E dans 50% des mélanomes).
PI3K/AKT/mTOR : RTK → PI3K → PIP3 → PDK1 → AKT → mTORC1 (protéosynthèse, prolifération, survie, métabolisme) + inactivation de BAD (anti-apoptose). PTEN = phosphatase anti-PI3K = suppresseur de tumeur. mTOR inhibé par la rapamycine (immunosuppresseur + anticancéreux).
Le vémurafénib cible BRAF V600E (mutation dans 50% des mélanomes) → inhibition de RAF → blocage de la voie MAPK → apoptose des cellules mélanomateuses. Réponse initiale souvent spectaculaire, mais résistance fréquente (réactivation de RAS ou d'autres voies). Association avec cobimétinib (inhibiteur MEK) retarde la résistance. Modèle de médecine de précision en oncologie.
🔵 Base
Signalisation via les récepteurs nucléaires — hormones stéroïdiennes
Les récepteurs nucléaires sont des facteurs de transcription intracellulaires activés par des ligands liposolubles (hormones stéroïdiennes, thyroïdiennes, vitamine D, acide rétinoïque). Ils traversent la membrane plasmique par diffusion simple, se fixent à leur récepteur intracellulaire → translocation nucléaire → liaison à des séquences HRE (Hormone Response Element) → activation ou répression de la transcription.
Les corticoïdes (cortisol, dexaméthasone) se fixent sur le récepteur glucocorticoïde (GR) → complexe GR-cortisol transloqué dans le noyau → liaison GRE → transcription de gènes anti-inflammatoires (annexine A1, GILZ) + répression des gènes pro-inflammatoires (NF-κB inhibé). Effets systémiques : immunosuppression puissante + hyperglycémie + catabolisme protéique + ostéoporose → surveillance indispensable en traitement prolongé. Les modulateurs sélectifs (SERM, SARM) tentent de dissocier les effets thérapeutiques des effets indésirables.
Les voies de signalisation reviennent dans chaque chapitre de physiopathologie. Mémorise : Gαs → AMPc → PKA (adrénaline, glucagon), Gαq → IP3/DAG → Ca2+/PKC (ACh, angiotensine II), RTK → RAS/MAPK (EGF, IGF). Et les toxines bactériennes qui piratent ces voies : cholérique = Gαs bloqué actif.
