Structure des nucléotides — les briques de l'ADN et de l'ARN
Un nucléotide = base azotée + sucre (pentose) + groupement phosphate. Les bases puriques (2 cycles) : adénine (A) et guanine (G). Les bases pyrimidiques (1 cycle) : cytosine (C), thymine (T, ADN seulement), uracile (U, ARN seulement). Sucre : désoxyribose (ADN, OH absent en 2') vs ribose (ARN, OH en 2').
Différences ADN / ARN
ADN : désoxyribose, bases ACGT, double brin (hélice droite B), stable.
ARN : ribose, bases ACGU (U remplace T), simple brin principalement, moins stable.
Liaisons phosphodiester : relient le OH-3' d'un nucléotide au phosphate du suivant → chaîne 5'→3'.
Appariements Watson-Crick : A=T (2 liaisons H), G≡C (3 liaisons H).
GC riche → Tm plus élevée (plus de liaisons H → plus stable à la chaleur).
À retenir
A=T (2 liaisons H) · G≡C (3 liaisons H) · %G = %C et %A = %T (règle de Chargaff)
Tm = température de fusion de l'ADN · GC ↑ → Tm ↑ · 1% GC ≈ +0,41°C
La PCR (réaction en chaîne par polymérase) exploite ces propriétés : dénaturation à 95°C (rupture liaisons H → séparation brins), hybridation des amorces à ~55-65°C (complémentarité A-T/G-C), extension à 72°C (Taq polymérase). Chaque cycle double la quantité d'ADN. Outil fondamental du diagnostic moléculaire (COVID-19, génétique, oncologie).
Double hélice de l'ADN et compaction chromatinienne
La double hélice B (Watson & Crick, 1953) : deux brins antiparallèles (5'→3' et 3'→5'), hélicoïdaux droites, un tour = 10 paires de bases = 3,4 nm. Les bases sont à l'intérieur (empilées = stacking), le sucre-phosphate à l'extérieur. Dans le noyau, l'ADN est compacté par les protéines histones.
Niveaux de compaction de la chromatine
Nucléosome : 147 pb d'ADN enroulées autour d'un octamère d'histones (H2A, H2B, H3, H4 × 2) + H1 (linker histone).
Fibre de 30 nm : solénoïde de nucléosomes.
Domaines en boucle : fibres ancrées sur une matrice protéique.
Chromosome métaphasique : compaction maximale (~10 000×). Euchromatine (active) vs hétérochromatine (inactive, condensée).
Les modifications des histones régulent l'accès à l'ADN : acétylation des Lys (HAT = histone acétyltransférase) → chromatine relaxée → transcription activée. Déacétylation (HDAC) → chromatine condensée → transcription réprimée. Les inhibiteurs de HDAC (vorinostat, romidepsine) sont utilisés comme anticancéreux → réactivation de gènes suppresseurs de tumeurs.
Réplication de l'ADN — mécanisme semi-conservatif
La réplication de l'ADN est semi-conservative (chaque brin sert de matrice, chaque molécule fille contient 1 brin parental + 1 brin néosynthétisé). Elle se fait en 5'→3' uniquement. Commence aux origines de réplication (ori), bidirectionnelle.
Acteurs clés de la réplication
Hélicase : déroule la double hélice au niveau de la fourche de réplication.
Primase : synthétise une amorce ARN (10-12 nt) — nécessaire car les ADN polymérases ne peuvent pas initier de novo.
ADN polymérase III (procaryote) / pol δ, ε (eucaryote) : synthèse 5'→3', activité de relecture (3'→5' exonucléase).
Brin continu : synthèse continue. Brin discontinu : fragments d'Okazaki + ADN ligase.
Télomérase : rallonge les télomères (ARN télomérase + transcriptase inverse).
À retenir
Réplication semi-conservative : 1 molécule parent → 2 molécules filles (chacune 50% parentale)
Exactitude : 1 erreur / 109 pb (correction par ADN pol + système MMR + BER + NER)
La télomérase est surexprimée dans ~85% des cancers → rallonge les télomères → immortalisation cellulaire. Dans les cellules somatiques normales, absence de télomérase → raccourcissement des télomères à chaque division → limite de Hayflick (~50 divisions) → sénescence. Les inhibiteurs de télomérase sont des cibles anticancéreuses en développement.
Transcription — synthèse de l'ARN
La transcription est la synthèse d'ARN à partir d'un brin matrice d'ADN, par l'ARN polymérase. Direction : 5'→3' sur le brin néosynthétisé (donc le brin matrice est lu 3'→5'). Chez les eucaryotes : transcription dans le noyau, puis maturation de l'ARNm (coiffe 5', queue poly-A, épissage des introns).
Types d'ARN et leurs fonctions
ARNm (messager) : code les protéines. Coiffe 5' (méthylguanosine) + poly-A 3' (stabilité + traduction).
ARNr (ribosomique) : composant structural et catalytique des ribosomes (ARNr 23S = ribozyme, activité peptidyl-transférase).
ARNt (transfert) : adaptateur codon-acide aminé (structure en trèfle, anticodon en boucle).
ARN non codants : miARN (~22 nt, régulation post-transcriptionnelle), siARN (interférence ARN, CRISPR), lncARN.
Le splicing alternatif permet à un seul gène de produire plusieurs protéines différentes en incluant ou excluant des exons. Le gène de la tropomyosine produit >10 isoformes selon les tissus. Le dérèglement du splicing est impliqué dans de nombreux cancers et maladies génétiques (ex : bêta-thalassémie par mutation du site d'épissage).
🟡 Importants
Traduction — le code génétique
La traduction est la synthèse de protéines par les ribosomes à partir du code porté par l'ARNm. Le code génétique est un triplet de bases (codon) = 1 acide aminé. Il est dégénéré (plusieurs codons pour un même AA) mais non ambigu (1 codon → 1 AA).
Propriétés du code génétique et codons importants
Codon initiateur : AUG (méthionine) → TOUJOURS le premier codon traduit → Met en N-terminal.
Codons stop : UAA, UAG, UGA — pas d'ARNt correspondant → arrêt de la traduction.
Dégénéré : 64 codons pour 20 AA + 3 stops → plusieurs codons/AA. La 3e base est la moins contrainte (wobble).
Universel : quasi-universel (mitochondries ont des codons légèrement différents).
Direction : AUG en 5' → traduction 5'→3' → protéine N→C.
À retenir
43 = 64 codons · 61 codons sens + 3 codons stop · Dégénéré mais non ambigu
AUG = Met = codon initiateur · UAA/UAG/UGA = stop · lecture 5'→3'
Les antibiotiques ciblant la traduction exploitent les différences entre ribosomes procaryotes (70S = 50S + 30S) et eucaryotes (80S = 60S + 40S). Tétracyclines → bloquent la fixation de l'ARNt sur la sous-unité 30S. Macrolides (érythromycine) → bloquent la translocation sur 50S. Cycloheximide → inhibe la translocation sur 80S (eucaryotes → toxique pour humain, usage antifongique uniquement).
🔵 Base
Mutations et réparation de l'ADN
Une mutation est un changement permanent de la séquence d'ADN. Types : substitution (ponctuelle), insertion/délétion (frameshift), amplification. Causes : erreurs de réplication, rayonnements UV (dimères de thymine), agents alkylants, désamination de la cytosine (→ uracile).
Systèmes de réparation de l'ADN
MMR (Mismatch Repair) : répare les mésappariements post-réplication. Déficit → syndrome de Lynch (cancer colorectal héréditaire).
NER (Nucleotide Excision Repair) : répare les lésions volumineuses (dimères UV). Déficit → xeroderma pigmentosum (photosensibilité extrême, cancers cutanés).
BER (Base Excision Repair) : répare les bases endommagées (désamination, oxydation).
NHEJ / HDR : réparation des cassures double brin (NHEJ = Non-Homologous End Joining, erreur-prone · HDR = Homologous Directed Repair, précise).
BRCA1/BRCA2 (Breast Cancer genes) sont des protéines clés de la réparation par HDR. Mutation germinale → déficit de réparation des cassures double brin → accumulation de mutations → cancer du sein et de l'ovaire. Les inhibiteurs de PARP (olaparib) exploitent la létalité synthétique : les cellules BRCA-mutées dépendent de PARP pour survivre → inhiber PARP → mort sélective des cellules tumorales.
ADN/ARN est souvent traité conjointement avec la biologie moléculaire. Mémorise le dogme central : ADN → (réplication) ADN, ADN → (transcription) ARN → (traduction) Protéine. Et les différences ADN/ARN : désoxyribose/ribose, T/U, double/simple brin, stable/instable.
