🔴 Incontournables
Qu'est-ce qu'une fonction ?
Une fonction f associe à chaque valeur de x (la variable) une valeur f(x). On dit que x est la variable et f(x) est l'image de x par f. Graphiquement, c'est une courbe dans un repère.
Vocabulaire de base
Domaine de définition : ensemble des valeurs de x pour lesquelles f(x) existe. Ex : f(x) = 1/x est définie pour tout x ≠ 0.
Image : f(x) est l'image de x. Ex : si f(x) = 2x+1, l'image de 3 est f(3) = 7.
Antécédent : x est un antécédent de f(x). Ex : 3 est un antécédent de 7 par f.
Parité : f paire si f(-x) = f(x) (symétrie axe y). f impaire si f(-x) = -f(x) (symétrie origine).
f(x) = x² est une fonction paire (f(-x) = (-x)² = x² = f(x)). Sa courbe est une parabole symétrique par rapport à l'axe des ordonnées. Son domaine de définition est ℝ tout entier.
La dérivée — qu'est-ce que c'est ?
La dérivée f'(x) d'une fonction f en un point x représente le taux de variation instantané — c'est la pente de la tangente à la courbe en ce point. Si f'(x) > 0, la fonction monte. Si f'(x) < 0, elle descend. Si f'(x) = 0, c'est un extremum (max ou min local).
Dérivées des fonctions de base à connaître absolument
f(x) = constante → f'(x) = 0
f(x) = x → f'(x) = 1
f(x) = xⁿ → f'(x) = n·xⁿ⁻¹
f(x) = √x → f'(x) = 1/(2√x)
f(x) = 1/x → f'(x) = -1/x²
f(x) = eˣ → f'(x) = eˣ (la seule fonction égale à sa dérivée !)
f(x) = ln(x) → f'(x) = 1/x
f(x) = sin(x) → f'(x) = cos(x)
f(x) = cos(x) → f'(x) = -sin(x)
Si f(x) = x³ - 3x, alors f'(x) = 3x² - 3. Les extremums sont là où f'(x) = 0 : 3x² - 3 = 0 → x = ±1. En x=1 : f'(x) passe de + à - → maximum local. En x=-1 : f'(x) passe de - à + → minimum local.
Règles de dérivation — combinaisons
Pour dériver une combinaison de fonctions, on utilise des règles. Les plus importantes pour le PASS sont la règle de la somme, du produit et de la composition.
Règles principales
(f + g)' = f' + g' : dérivée d'une somme = somme des dérivées
(k·f)' = k·f' : la constante sort
(f·g)' = f'g + fg' : règle du produit
(f/g)' = (f'g - fg') / g² : règle du quotient
(f∘g)' = g' · f'(g(x)) : règle de la chaîne (composition)

Cas fréquents de la règle de la chaîne :
— (eᵘ)' = u'·eᵘ
— (ln(u))' = u'/u
— (uⁿ)' = n·u'·uⁿ⁻¹
Dériver f(x) = exp(2x) : c'est une composition avec u = 2x → u' = 2. Donc f'(x) = 2·exp(2x). Dériver g(x) = ln(x²+1) : u = x²+1 → u' = 2x. Donc g'(x) = 2x/(x²+1).
🟡 Importants
Sens de variation et tableau de signes
Pour étudier les variations d'une fonction, on calcule f'(x) et on étudie son signe. Là où f'(x) > 0, f est croissante. Là où f'(x) < 0, f est décroissante. Les points où f'(x) = 0 sont les extremums potentiels.
Méthode — tableau de variations
1. Calculer f'(x)
2. Résoudre f'(x) = 0 → trouver les points critiques
3. Étudier le signe de f'(x) entre les points critiques
4. Dresser le tableau de variations

Extremum local : si f'(x) change de signe en a, alors f(a) est un extremum. Si f' passe de + à - → maximum. Si f' passe de - à + → minimum.
f(x) = x² - 4x + 3. f'(x) = 2x - 4 = 0 → x = 2. Pour x < 2 : f'(x) < 0 (décroissante). Pour x > 2 : f'(x) > 0 (croissante). Donc f(2) = 4 - 8 + 3 = -1 est un minimum. En PASS, ce type d'analyse sert pour les équilibres chimiques et les courbes dose-réponse.
🔵 Base
Limites — comportement aux extrêmes
La limite d'une fonction en un point ou à l'infini décrit son comportement. En PASS on utilise surtout les limites à l'infini (comportement asymptotique) et en certains points (discontinuités).
Limites usuelles à connaître
Polynômes : domination par le terme de plus haut degré. Ex : lim(x→+∞) de x³ - x = +∞
Fonctions exponentielles : eˣ → +∞ plus vite que tout polynôme. e⁻ˣ → 0
Logarithme : ln(x) → +∞ mais très lentement. ln(x) → -∞ quand x→0⁺
Formes indéterminées : ∞/∞, 0/0, ∞-∞ → factoriser ou utiliser la règle de L'Hôpital
Croissances comparées : eˣ >> xⁿ >> ln(x) (ordre de grandeur quand x→+∞)
En pharmacologie : la concentration d'un médicament C(t) = C₀·exp(-kt) tend vers 0 quand t→+∞ (élimination progressive). La limite est 0 — le médicament est entièrement éliminé à terme. Ce comportement asymptotique est fondamental pour comprendre la pharmacocinétique.
Les 3 dérivées à avoir par cœur : (xⁿ)' = n·xⁿ⁻¹, (eˣ)' = eˣ, (ln x)' = 1/x. Et la règle de la chaîne : (eᵘ)' = u'·eᵘ et (ln u)' = u'/u. Ce sont les outils les plus utilisés en physique, chimie et stats du PASS.
